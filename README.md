# 概要

WEBカフェラテアートは、マウスやタッチ操作でコーヒーにミルクを注ぎ、本物のようなラテアートを作成できるWebアプリケーションです。Navier-Stokes方程式に基づいた流体シミュレーションを実装しており、リアルな流体の動きを再現しています。
